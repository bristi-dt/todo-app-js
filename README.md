# Todo App (Vanilla JavaScript)

A simple and interactive Todo application built using HTML, CSS and Vanilla JavaScript.  
This project allows users to manage tasks by adding, editing, deleting and marking them as completed.

## Features

✔ Add new todos  
✔ Edit existing todos  
✔ Delete tasks  
✔ Mark tasks as completed  
✔ Only one todo can be edited at a time  

## Tech Stack

HTML  
CSS  
JavaScript
